## 概念

### 集合含义及表示

- 含义
  - N：自然数集  
  - Z：整数集  
  - Q：有理数集  
  - R：实数集
  - 空集、子集、真子集

- 常用表示法
  - 列举法：C = { 太湖, 洪泽湖 }  
  - 描述法：c = { x | x>32 }

- 简单示例：
  - 小于10的所有有理数集合：{ x∈Q, x<10 }  
  - 所有偶数集合：{ x | x=2n, n∈Z }
