## 油腻的中年人，请对自己好一点

《不想做成年人？满满的回忆》，一群油腻中年大叔给自己放了一天假，六一一过，还得回到艰难的生活中来。
人到中年，压力特别大：老人，小孩，工作... 这个时候，心态很容易奔溃。心态的调节，异常重要，这里分享一些自己的思考。

1. 要花额外的时间锻炼身体。

   我最深刻的感触是熬夜，刚毕业那会，连续熬3个晚上写code都没事；而现在，利物浦的比赛还没开始，就扛不住了。

   另一个感触是打篮球，上大学那会，连场PK打一个小时都不累；而现在，上篮，我躲，拉杆，我去，腰扭了。

   >画外音：于我，健康排在第一位，任何事情，都不要拿健康去做交换。

2. 多花一些时间陪家人。

   别再和父母起过多的争执，别再尝试改变ta们的生活习惯了，不涉及原则的情况下，只要老人家开心就行。

   多疼疼老婆，把挣的钱花在她身上，人最大的成功就是经营好家庭，婚姻经营不好，做什么都难以成功。老婆开心了，你才可能开心：happy wife, happy life. 这是我最大的心得。

   >画外音：于我，最爱老婆，其次是女儿，女儿已经有一个最爱她的妈妈。

   尽可能多的陪陪孩子，你再不陪ta，ta就长大了。父爱如山，不是指父亲坐在电脑前一动不动。你能给到孩子最珍贵的，不是最华丽的玩具，而是陪伴ta的时间。

3. 旅行？阅读？培养几个自己的爱好。

   到世上一趟不容易，人生本身就是“体验”的集合：

   - 空间维度，大千世界，多出去走走，被大自然的鬼斧神工所震撼
   - 时间维度，历史长河，多阅读看看，学习前人多年的智慧沉淀

   除了上班，陪家人，尽量要有几项自己的兴趣爱好。

   >画外音：于我，是写作，于你呢？

4. 控制住情绪，做一个成年人，并对自己好一点。

   不用过于在意别人对你的看法和评价，你不能让所有人满意，你不能控制所有人的想法，ta们于你，只是外人，活出自己，才是正道。如果有人称赞你年轻帅气，也不要沾沾自喜，微笑着表示感谢，人家是礼貌。

   >画外音：应该在意的是，你的亲人，与你信任的人，对你的看法，并对他们解释一些误会。

   不要在背后评论别人（只讲好话），不要参与流言蜚语的传播，更不要搬弄是非；可以有所保留，但一定不要撒谎，这是起码的修养。

   别喝太多酒，更别喝醉，你的素质和底牌容易在醉后一览无余；少抽点烟；少熬点夜。

   减少和他人争执，除了气到自己，其实你赢不了什么；正确的经验，提醒到位已经仁至义尽，不要强迫别人认同你。

   穿好一点的内裤，如果经济允许，买一辆好一点的车，油腻的中年人，请对自己好一点。

**总结：**

- 锻炼身体，早点睡
- 多花一点时间陪家人
- 旅旅游，看看书，培养几项兴趣爱好
- 少争执，少生气，对自己好一点
