scrollLeft:设置或获取位于对象左边界和窗口中目前可见内容的最左端之间的距离

## 类型检测

1. typeof

   语法：

   ```js
   typeof(表达式)
   typeof 变量名
   ```

   返回值：

   ```sh
   1. 'undefined'  # 未定义的变量或值
   2. 'boolean'    # 布尔类型的变量或值
   3. 'string'     # 字符串类型的变量或值
   4. 'number'     # 数字类型的变量或值
   5. 'object'     # 对象类型的变量或值，或者null
   6. 'function'   # 函数类型的变量或值
   ```

   将null作为object类型处理是js历史遗留问题。
