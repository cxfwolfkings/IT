## 树的实现

3种方法：

1. 双亲表示法
2. 孩子表示法
3. 孩子兄弟表示法

### 孩子兄弟表示法

```C
typedef struct TreeNode *PtrToNode;
struct TreeNode
{
  ElementType Element;
  PtrToNode FirstChild;
  PtrToNode NextSibling;
}
```

```java
class TreeNode {
  Object Element;
  TreeNode FirstChild;
  TreeNode NextSibling;
}
```
