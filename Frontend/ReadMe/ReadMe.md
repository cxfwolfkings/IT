## 谷歌浏览器出现“错误代码：ERR_UNSAFE_PORT”的解决办法

出错原因：谷歌浏览器（FF浏览器也有）对一些特殊的端口进行了限制

解决方法：最简单的办法就是直接修改搭建项目的端口号，避开这些谷歌限制的端口号。
