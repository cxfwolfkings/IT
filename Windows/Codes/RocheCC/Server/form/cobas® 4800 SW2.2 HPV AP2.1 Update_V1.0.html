关于 cobas® 4800系统HPV程序升级至V2.1版本

尊敬的罗氏用户：

我公司技术人员将会为您的cobas® 4800系统目前的 HPV项目程序（HPV Analysis Package V2.0.1) 升级至最新的 V2.1版本。此新版本HPV Analysis Package V2.1 项目程序，将更加匹配当前使用的cobas® 4800系统软件SW2.2的操作平台，本次升级对系统的性能参数与规格不会产生任何变更，升级过程不会产生任何安全相关的风险，仅是改善与提升HPV检测项目的可靠性、稳定性。

我们强烈建议您进行本次cobas® 4800系统HPV项目程序Analysis Package V2.1 的升级。

此次HPV项目程序（Analysis Package V2.1）升级，主要新的功能和特点包括：

  • 优化改进深孔板DWP耗材的设计，更换深孔板DWP的耗材款式，消除深孔板孔位间的毛细管效应，降低交叉污染的风险；原旧款式的深孔板耗材Extraction Plate (DWP)2.0 mL代替，新的HPV项目程序AP V2.1只识别探测和使用新款式的深孔板DWP,原旧款式的深孔板DWP将停止使用。
  • 优化了拾取吸头时的算法程序（减少吸头的浪费）；
  • 改进Recovery工作流程；
  • 改进MGP样本处理方法；
  • 改进无效结果累计计数器的功能；
  • 优化操作软件报警或标记提示信息；
  • 优化操作界面等；
